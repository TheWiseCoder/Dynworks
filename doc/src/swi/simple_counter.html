simple_counter.pl -- A global, stack-independent, simple counter

This is an implementation of a global, stack-independent, simple counter for the SWI-Prolog platform. It is intended for use with integer values, only. However, for performance reasons this is not enforced anywhere in the code. Consequences of non-compliance with this rule are unpredictable.
